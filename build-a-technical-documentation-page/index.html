Introduction

HTML stands for HyperText Markup Language and is the foundation of the web.

It provides the structure and meaning of web content.

Basic HTML Structure

An HTML document follows a standard structure that browsers understand.

This structure ensures proper rendering and accessibility.

<!DOCTYPE html> <html> <head></head> <body></body> </html>
Common HTML Elements

HTML provides many elements to display text, images, and links.

These elements help organize and present content clearly.

  • Paragraphs
  • Headings
  • Links
  • Images
  • Lists
<p>This is a paragraph</p> <a href="#">This is a link</a>
Forms and Inputs

Forms allow users to send data to a server.

Inputs collect information such as names, emails, and passwords.

  • Text input
  • Email input
  • Password input
<form> <input type="text"> </form>
Semantic HTML

Semantic elements describe the meaning of content.

They improve accessibility and SEO.

  • header
  • nav
  • main
  • section
  • footer
<main> <section>Content</section> </main>
Conclusion

HTML is essential for building structured and accessible web pages.
